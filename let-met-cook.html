Let Me Cook

A 2D cooking mobile game made using Unity inspired by Cooking Mama.
I worked on the assets integration and the development of some mini-games. it was the first time I worked on a mobile game (and Unity) so I experimented with the gyroscope capabilities.

We made it in about 60 hours in 2023.
